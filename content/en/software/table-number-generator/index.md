A tool for creating printable table numbers in the form of triangular cones.

## About the Application

The Table Number Generator allows you to create printable table numbers for bridge tournaments. The numbers are designed as triangular cones that can be folded and easily placed on tables.

## Usage

The application is available at: [Table Number Generator](/apps/table-cones/)

### Basic Features

- **Number Range**: Set the starting and ending table number
- **Triangle Size**: Adjustable triangle side length (30-120 mm)
- **Custom Text**: Optionally add custom text below the number (e.g., "OPEN", "NOVICE")
- **Automatic Layout**: The application automatically calculates the optimal number of cones per page and spacing between them

### Advanced Options

- **Font Selection**: Choose from several preset fonts or enter a custom font name
- **Font Size Adjustment**: Fine-tune the number size
- **Text Position**: Adjust the vertical position of text within the triangle
- **Cones per Page**: Manually set the number of cones on one A4 page
- **Gap Between Cones**: Adjust spacing between individual cones

## Usage Tips

- **Colored Paper**: If using colored paper, consider adding the color name to the custom text for better accessibility (e.g., "RED" or "BLUE")
- **Triangle Size**: Larger triangles are more visible but take up more space on the page

## Technical Details

The application generates SVG vector images that are optimized for printing. Each cone consists of three triangles that can be folded into a three-dimensional shape.
